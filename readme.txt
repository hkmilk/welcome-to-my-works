== Description ==

このテーマは、ポートフォリオ・ブログ用途を想定したオリジナルテーマです。
HTML5, CSS3, WordPressのFLOCSS設計で構築されています。

== Installation ==

1. このフォルダを `wp-content/themes/` にアップロード
2. 管理画面で有効化してください
